## Todo List App

A simple Todo List application built with **JavaScript**, organized using the **MVC pattern** and bundled with **Webpack**.

The goal of this project is to practice structuring JavaScript applications, separating logic from UI, and working with modular code.



## Features

* Create projects
* Add todos inside projects
* Mark todos as complete
* Delete todos
* Organize tasks by project
* Clean MVC structure
* Bundled with Webpack
